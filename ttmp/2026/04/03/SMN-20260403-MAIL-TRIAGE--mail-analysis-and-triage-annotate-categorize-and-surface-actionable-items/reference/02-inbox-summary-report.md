## 📋 Annotation DB Stats

All annotations use `source_label: mail-triage-v1` and `created_by: pi-agent` for traceability.

```sql
-- Query all annotations
SELECT tag, COUNT(*) FROM annotations GROUP BY tag ORDER BY COUNT(*) DESC;

-- Query unsubscribe candidates
SELECT g.name, m.target_id FROM target_groups g
JOIN target_group_members m ON m.group_id = g.id
WHERE g.name = 'Unsubscribe Candidates';

-- Query personal contacts  
SELECT g.name, m.target_id FROM target_groups g
JOIN target_group_members m ON m.group_id = g.id
WHERE g.name = 'Personal Contacts';

-- Find recent messages from personal contacts
SELECT id, substr(internal_date,1,10), sender_email, substr(subject,1,70)
FROM messages m
WHERE EXISTS (SELECT 1 FROM annotations a WHERE a.target_type='sender' AND a.target_id=m.sender_email AND a.tag='personal')
ORDER BY internal_date DESC LIMIT 20;
```
